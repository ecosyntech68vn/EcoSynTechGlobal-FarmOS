## 🎯 SỰ CỐ THƯỜNG GẶP

### 1. Bị hack tài khoản
**Dấu hiệu:** Login từ IP lạ, password không đổi được

**Xử lý:**
```
1. Đổi password ngay trên app
2. Liên hệ support để lock phiên cũ
3. Kiểm tra lịch sử đăng nhập
4. Bật 2FA nếu chưa có
```

### 2. Thiết bị bị tấn công
**Dấu hiệu:** Thiết bị offline, phát nhiệt cao, đèn nhấp nháy bất thường

**Xử lý:**
```
1. Rút nguồn điện thiết bị NGAY
2. Chụp ảnh hiện trạng
3. Gửi ID thiết bị cho support
4. Đợi hướng dẫn reset
```

### 3. Dữ liệu bị xóa/mã hóa
**Dấu hiệu:** Không truy cập được data, thấy thông báo lạ

**Xử lý:**
```
1. KHÔNG tắt server
2. KHÔNG restart
3. Chụp ảnh màn hình lỗi
4. Liên hệ support NGAY
5. Có thể khôi phục từ backup
```

### 4. Bị tấn công DDOS
**Dấu hiệu:** Web không vào được, chậm bất thường

**Xử lý:**
```
1. Thông báo ngay cho support
2. Có thể tạm tắt tính năng không cần thiết
3. Đợi support block IP tấn công
```
