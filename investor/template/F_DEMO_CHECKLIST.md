# EcoSynTech FarmOS - Demo Checklist
## Checklist Demo cho Sales

---

## 1. CHUẨN BỊ TRƯỚC DEMO

### 1.1 Thiết bị demo
```
□ Controller (đã cài đặt sẵn)
□ Cảm biến mẫu:
  □ ST30 (nhiệt độ)
  □ Soil Moisture (độ ẩm đất)
  □ DHT22 (nhiệt/độ ẩm)
  □ BME280 (nhiệt/độ ẩm/áp suất)
□ Relay mẫu (có LED indicator)
□ Nguồn 12V
□ Bộ kit lắp đặt demo
```

### 1.2 Tài liệu
```
□ Product one-pager (in sẵn)
□ Bảng giá
□ Case studies / testimonials
□ QR code App download
□ Video demo (trên điện thoại)
```

### 1.3 Check App
```
□ Đã cài đặt App mới nhất
□ Đã login với tài demo
□ Dashboard hiển thị dữ liệu
□ Chức năng bật/tắt relay hoạt động
□ Cảnh báo Zalo đã kết nối
```

### 1.4 Check thiết bị
```
□ Controller bật nguồn, đèn sáng
□ Tất cả cảm biến kết nối
□ Relay phản hồi bật/tắt
□ Wifi kết nối ổn định
```

## 7. PHỤ LỤC

### 7.1 Quick Demo Script (2 phút)
```
1. "Em sẽ cho anh/chị xem thiết bị thật"
2. "Đây là controller, đo nhiệt độ, độ ẩm..."
3. "Em sẽ bật máy bơm cho anh/chị xem"
4. "Anh/chị có thể bật/tắt trên điện thoại này"
5. "Anh/chị thấy dữ liệu cập nhật liên tục"
6. "Em tính giá cho anh/chị nhé?"
```

### 7.2 Objection Handling during Demo
```
Khách nói "đắt":
→ "Anh/chị tính thử ROI nhé. Tiết kiệm 30% nước = 150K/tháng.
   Hoàn vốn trong 1 năm thôi ạ."

Khách nói "khó dùng":
→ "Anh/chị thấy không ạ? Bấm nút này là bật rồi.
   Em dạy 5 phút là biết dùng."

Khách nói "để xem xét":
→ "Dạ, anh/chị dùng thử 1 tuần không tiền nhé.
   Không phải mua đâu ạ."
```
