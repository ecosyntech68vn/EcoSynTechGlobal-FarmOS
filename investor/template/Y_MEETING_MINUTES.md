## 2. MEETING FORMAT

### Standard Agenda

```
┌────────────────────────────────────────────────────────────┐
│              MEETING AGENDA                               │
├────────────────────────────────────────────────────────────┤
│  1. PREVIOUS ACTION ITEMS (Review)                       │
│  2. METRICS UPDATE                                       │
│  3. PROGRESS REPORTS                                     │
│  4. BLOCKERS & ISSUES                                    │
│  5. NEW BUSINESS                                        │
│  6. ACTION ITEMS (New)                                   │
│  7. NEXT MEETING DATE                                    │
```

## 3. ACTION TRACKING

### Action Item Format

| ID | Task | Owner | Due | Status |
|----|------|-------|-----|--------|
| A1 | Task description | Name | Date | ⬜ |

### Follow-up Process

```
Step 1: Assign action item
Step 2: Set due date
Step 3: Review in next meeting
Step 4: Update status
Step 5: Close or escalate
```
