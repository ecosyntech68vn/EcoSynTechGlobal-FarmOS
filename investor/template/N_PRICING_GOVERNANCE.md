## 4. APPROVAL AUTHORITY

### Discount Approval Levels

| Discount Level | Approval | Documentation |
|----------------|----------|----------------|
| <5% | Sales lead | Email approval |
| 5-10% | Manager | Written request |
| 10-20% | CEO | Business case |
| >20% | Board | Formal proposal |

### Price Exception Process

```
Step 1: Request with justification
Step 2: Manager review (1-2 days)
Step 3: Finance verification
Step 4: CEO approval
Step 5: Documentation filed
Step 6: Customer communication
```

## 5. PRICING COMPLIANCE

### Prohibited Practices

```
┌────────────────────────────────────────────────────────────┐
│           PRICING COMPLIANCE RULES                          │
├────────────────────────────────────────────────────────────┤
│                                                              │
│  FORBIDDEN:                                                 │
│  • Unapproved discounts                                     │
│  • Bribing procurement staff                            │
│  • Fake competitor quotes                                 │
│  • Price matching without process                       │
│  • Hidden fees                                            │
│  • Misleading pricing                                    │
│                                                              │
│  REQUIRED:                                                │
│  • Written quotes only                                   │
│  • Invoice matches quote                                  │
│  • Clear terms and conditions                          │
│  • Customer signature on custom                        │
│                                                              │
└────────────────────────────────────────────────────────────┘
```
