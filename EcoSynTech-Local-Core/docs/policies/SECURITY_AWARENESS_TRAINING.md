# Security Awareness Training Program

**Document ID:** SEC-TRN-001  
**Version:** 1.0  
**Effective Date:** 2026-04-24  
**ISO 27001:2022 Controls:** A.6.3  

---

## 1. Purpose

This document outlines the security awareness training program for all employees and end-users (farmers) who interact with the EcoSynTech Farm OS system. The program ensures that all personnel understand their security responsibilities and can identify potential security threats.

---

## 2. Target Audience

| Audience | Training Type | Frequency |
|----------|---------------|-----------|
| Developers/IT Staff | Technical Security | Quarterly |
| Operations Team | Operational Security | Quarterly |
| Farmers/End Users | Basic Security Awareness | Onboarding + Annual |
| Management | ISMS Overview | Annual |

---

## 3. Training Modules

### Module 1: Basic Security Awareness (All Users)
- Password security best practices
- Recognizing phishing attempts
- Physical security for devices
- Reporting security incidents
- Data classification basics

### Module 2: System-Specific Security (Farmers)
- Secure login practices
- Device pairing security
- Sensor data handling
- Alert management
- Data privacy basics

### Module 3: Technical Security (IT/Dev)
- Secure coding practices
- Git security and secrets management
- API security best practices
- Incident response procedures
- Compliance requirements (ISO 27001)

### Module 4: Management Security (Managers)
- ISMS overview
- Risk assessment basics
- Security policies and compliance
- Incident escalation procedures
- Audit preparation

---

## 4. Training Schedule

| Activity | Frequency | Responsible Party |
|----------|-----------|-------------------|
| New Employee Onboarding | Before system access | HR / IT Lead |
| Quarterly Technical Training | Every 3 months | Security Team |
| Annual Refresher Training | Every 12 months | Security Team |
| Post-Incident Training | As needed | Security Team |

## 6. Assessment

- **Quiz:** 10 questions after each training module (pass: 80%)
- **Phishing Simulation:** Quarterly simulated phishing tests
- **Compliance Tracking:** Training completion rates logged in HR system

---

## 7. Records

Training attendance and completion records will be maintained for a minimum of 3 years in accordance with ISO 27001:2022 A.5.31 (Records of management).

| Record Type | Retention | Storage |
|-------------|-----------|---------|
| Attendance Lists | 3 years | HR System |
| Assessment Results | 3 years | HR System |
| Certificates | 5 years | HR System |
