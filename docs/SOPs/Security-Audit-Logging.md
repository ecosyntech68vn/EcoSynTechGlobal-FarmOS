SOP: Security Audit & Logging

Goal
- Maintain auditable security controls and logs to support ISO 27001 and regulatory needs.

Scope
- All authentication, RBAC, API access logs, and security events.

Roles
- Security, DevOps, Compliance.

Audit & Logging
- Enable/maintain access logs, API usage logs, and security events.
- Rotate logs; protect log integrity; store logs securely offsite if needed.
- Regular security reviews and vulnerability scanning results should be captured as evidence.

Data Privacy & Retention
- Data retention rules; data minimization.

Testing & Validation
- Periodic audit checks; verify log completeness; sample data matches actions.
