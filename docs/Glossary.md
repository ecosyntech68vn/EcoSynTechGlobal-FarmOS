Glossary

- ISO 27001: Information Security Management System standard.
- 5S: Sort, Set in Order, Shine, Standardize, Sustain.
- PDCA: Plan-Do-Check-Act continuous improvement loop.
- 5W1H: What, Why, Who, Where, When, How; problem solving framework.
- API: Application Programming Interface (endpoints under /api).
- Dashboard: Web UI pages for monitoring and management.
- ETL: Extract-Transform-Load (data processing pattern).
- SoA: Statement of Applicability; ISO control mapping.
- RBAC: Role-Based Access Control.
- MFA: Multi-Factor Authentication.
- OSS: Open Source Software.
- CI/CD: Continuous Integration/Continuous Deployment.
- SOP: Standard Operating Procedure.
- ML: Machine Learning; Lightweight bootstrap vs. large model.
- Mock: Test doubles used for isolated testing.
