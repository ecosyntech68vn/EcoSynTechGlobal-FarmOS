Asset Register (Skeleton)

- Asset ID: unique identifier
- Asset Name: user friendly name
- Asset Type: Hardware | Software | Data | Document
- Owner: person responsible
- Classification: Public | Internal | Confidential
- Location/Host: where asset resides
- Status: Active | Inactive | Archived
- Retention: retention period (years)
- Security Controls: encryption at rest, access controls, backups
- Source/Acquisition Date: when asset was acquired
- Evidence: links to supporting docs or policy references
- Notes: any additional context

Notes
- This is a living document; update periodically as assets change or new assets are added.
