ISO27001 SoA Template (Skeleton)

- Control: A.x.y (e.g., A.8.1 Access Control Policy)
- Policy Owner: name/email
- Status: Implemented / Not Applicable / Partial
- Rationale: brief reason for status
- Evidence: list of artifacts (policy docs, logs, runbooks, tests, etc.)
- Additional Notes: remediation plan and due date
